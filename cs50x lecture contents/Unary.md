#CountingSystem

We use technology daily but in the of the all these days, underneath of it, everything that really exists is zeros and ones.
But no, the unary, or base-1, system, is not happening in the background of your tasks, it's not the [[Binary]] system yet.
The unary system is the simplest numeral system to represent natural numbers.

For instance, for you represent the number 32 in this system, you'll need to repeat one single symbol (in this case the symbol can "1", but it can be any symbol) thirty-two times, and that's because you only have one notation to use.
Out of the computer, if each finger of your right hand was +1 to the final count, you'd have 5 in the end.
Or, provided that, the symbol in question could be "|", if repeated 9 times each time a number will be added to the counter, ||||||||| = 9.

This is one of limits using the unary/base-1 system.
It is an additive, not a posicional, [[counting system]].
